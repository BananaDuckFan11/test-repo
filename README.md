# Test Repo

A quick application with a Python web server.

## Getting Started

### Installation

```bash
pip install -r requirements.txt
```

### Running the Server

```bash
python app.py
```

The server will start on `http://localhost:8000`

## Project Structure

- `app.py` - Main Python web server file
- `requirements.txt` - Python dependencies

## Requirements

- Python 3.7 or higher
